# Présentation 

## 1. Les étas du moi 

- concept de l’Analyse Transactionnelle
- initié par Éric Berne

![](../image/diagEM2.webp)\
_source : [analysetransactionnelle.fr](https://analysetransactionnelle.fr/p-Les_Etats_du_Moi)_


### 1.1. Parent

-  comportements de modèles parentaux (à l'image de ses propres parents)
- 2 types :
    - parent directeur
    - parent protecteur

### 1.2. Adult

- comportement basé de l'analyse objective de l'environement -> adaptation

### 1.3 Enfant

- comportements de modèles de l'enfant, vécu personel d'autrefois
- 3 types : 
    - enfant soumis
    - enfant libre
    - enfant rebelle (état intermédiaire)

## 2. La classification MBTI

- publié par Katherine Cook Briggs

Regroupe 16 types de personnalité differentes

![](../image/mbti1.png)\
_source : [wikipedia](https://fr.wikipedia.org/wiki/Myers_Briggs_Type_Indicator)_

- extraversion (E)-introversion (I) ;
- sensation (S)-intuition (N) ;
- pensée (T)-sentiment (F) ;
- jugement (J)-perception (P).
