# A la découverte d'Orange Pi !

Dans cet article nous allons partir à la découverte des mini ordinateurs avec le Orange Pi CM4 pour créer un ordiateur de breau sur linux.

![Orange Pi](../../image/orpi.jpg "Orange Pi")
_source : personnelle_

## Carte 

Nous possédons un  <a href="http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/details/Orange-Pi-CM4-2.html" target="_blank"> Orange Pi CM4 Based Board</a> ainsi qu'un <a href="http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/details/Orange-Pi-CM4-1.html" target="_blank">Orange Pi Compute Module 4</a>

Voici des <a href="http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/service-and-support/Orange-Pi-CM4-1.html" target="_blank">ressources</a>

## Installation
Sur une carte mémoire micro SDHC de 32 Gb, j'installe le système d'exploitation.


{{% columns %}}
- ![Rufus](../../image/rufusOrange.png "Rufus")
_source : personnelle_

- J'utilise <a href="https://rufus.ie/fr/" target="_blank">Rufus</a> pour flashé l'OS sur la carte SD

{{% /columns %}}

{{% columns %}}
- ![instal](../../image/opinstall.jpg "install")
_source : personnelle_

- Après avoir branché un écran, un clavier, une sourie et enfin l'alimentation. L'installation ce lance directement. Nous n'avons pratiquement rien à faire
