# Orca

Aujourd'hui nous allons nous interressé à <a href="https://100r.co/site/orca.html" target="_blank">Orca</a>. C'est un langage deprogrammation 
ésotérique en deux dimensions permettant de créer des mélodis

Orca n'est pas un synthetiseur, donc pour avoir du son il faut en ajouter un. Pour ma part j'ai choisi <a href="https://wiki.xxiivv.com/site/pilot.html" target="_blank">Pilot</a> ainsi que <a href="https://algosail.itch.io/cassetter" target="_blank">Cassetter</a> qui fait le lien entre les deux, comme un câble.


<a href="https://www.tobias-erichsen.de/software/loopmidi.html" target="_blank">LoopMidi</a>


Voici  <a href="https://metasyn.srht.site/learn-orca/" target="_blank">un lien</a> pour l'essayer en ligne sans installer de logiciel.
